# Algo Products

Real-world mini products built around LeetCode algorithm patterns.
Each product is a complete loop: **spec → build → publish**.

## Why?

AI makes it dangerously easy to skip the hard parts of engineering.

I'm building these tiny products as a way to strengthen my problem solving and tackle leet code style algorithms in a fresh way. Each product will serve as a pitch to implement and learn challenging concepts, which I feel is missing in the time of AI. 

I'll also be posting a dev log for each product along side this repo on my [substack](https://substack.com/@builtbysid), documenting the journey.

## The Rule

Build the data structure and core algorithm in a blank file, no AI.
Everything else — UI, wiring, article — AI is fair game.
